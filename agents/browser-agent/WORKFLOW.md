# WORKFLOW — browser-agent

## Цель
Поднять отдельную рабочую зону под браузерную автоматизацию с Gologin и Playwright внутри OpenClaw.

## Архитектура
- отдельная папка: `agents/browser-agent/`
- отдельный skill: `skills/gologin-automation/`
- отдельная agent/session: `browser-agent`
- браузерная автоматизация через Playwright
- профильные сессии браузера через Gologin

## Поток работы
1. Установить Gologin на текущую машину.
2. Создать или импортировать профиль браузера.
3. Войти вручную в нужные аккаунты.
4. Сохранить профиль и сессию внутри Gologin.
5. Запустить профиль с debugging endpoint.
6. Подключить Playwright к запущенному браузеру.
7. Выполнять автоматизацию в видимом режиме.

## Правила
- по умолчанию без скриншотов
- при риске большого расхода токенов — стоп и предупреждение
- логины / 2FA / оплаты / подтверждения — только под контролем человека
- повторные входы заменять повторным использованием профиля

## Ближайшие шаги
- проверить способ установки Gologin на Linux Mint / Ubuntu-подобную систему
- определить где хранить профили
- подготовить запуск Playwright через подключение к живому браузеру
- затем поднять отдельную session для browser-agent
