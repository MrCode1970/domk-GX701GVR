# INSTALL — Gologin on this machine

## Что подтверждено
- Linux Mint поддерживается официально
- Gologin для Linux поставляется как AppImage
- для автоматизации есть официальный SDK и запуск профиля с endpoint для Playwright/Selenium

## Рабочий путь
1. Скачать Gologin AppImage
2. Сделать файл исполняемым
3. При необходимости установить `libfuse2`
4. Запустить Gologin
5. Войти в аккаунт вручную
6. Создать/импортировать профиль
7. Проверить запуск профиля и endpoint
8. Подключить Playwright к живому профилю

## Что нужно проверить локально
- есть ли `libfuse2`
- куда лучше положить AppImage
- где хранить профили и заметки по ним

## Следующий практический шаг
Скачать и запустить Gologin на этой машине.
