# Правило работы в `#fox-chat` и тредах

## Одна строка
**Чат = вход. Тема = тред. Тред = контекст. Итог = архив/память/KB.**

## Три правила

### 1. `#fox-chat` — вход и быстрые вопросы
Коротко. Текущее. Несвязанное. Быстро.

### 2. `тема:` — новая рабочая линия
Если начинается отдельная работа, помечаем как:
```
тема: Название работы
```
Дальше это **автоматически уходит в отдельный тред**.

### 3. Один тред = один контекст
Можно вести несколько тредов параллельно.  
Между ними переключаться в любом порядке.  
Контекст сохраняется внутри треда.

---

## Когда создаём тред?

Если:
- тема дольше 3–5 сообщений
- имеет отдельную цель
- к ней надо возвращаться
- мешает основному чату

**Помечаем `тема:` → тред создаётся.**

---

## Когда тред готов?

Если:
- есть результат
- или решено не делать
- или тему честно закрыли

**Решаем: архив / память / KB?**

---

## Формула

```
чат рождает
  ↓
тема создаёт тред
  ↓
тред ведёт работу
  ↓
результат → архив / память / KB
```

---

## Это всё

Больше ничего не надо помнить.  
Остальное следует из этого.
